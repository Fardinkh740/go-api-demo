# Go API Demo – Tests, Logs & LLM Analysis 🚀

Dieses Projekt zeigt ein einfaches Beispiel für:
- Aufbau einer REST API in **Go**
- Automatisierte Tests mit `go test`
- Farbiges Logging (ANSI)
- KI-gestützte Log-Analyse (LLM / ChatGPT)

## 📦 Setup

1️⃣ Repository klonen oder lokal öffnen  
2️⃣ Go installieren (Version ≥ 1.21)  
3️⃣ Modul initialisieren:
```bash
go mod init go-api-demo
